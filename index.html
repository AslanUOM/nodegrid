Welcome to NodeGrid

Nodegrid is a Mobile Backend-as-a-Service which helps you to build rich mobile applications without writing any sever-side code. NodeGrid is built upon NodeJs, therefore it performs faster and lighter. This is embedded with many cool features and all those services are securely exposed via REST APIs
